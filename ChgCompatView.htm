"Upgraded" to Internet Explorer 11?

 Has your browser been "upgraded" to Internet Explorer Version 11?

If so, it may not be displaying this system's pages correctly, and some functions may not work properly.

You can follow the steps below to correct the problem. This is a one-time change for your browser.

To check the version of Internet Explorer you are using, click Help in the Menu bar, then About Internet Explorer.

If the About box does not say "Internet Explorer 11," you don't need to bother with this change.

But if the About box says "Internet Explorer 11," follow these steps:

  1. In the Menu bar, click Tools > Compatibility View settings.
  2. In the Compatibility View Settings box, next to Add this website, you should see "usps.gov" highlighted. Click the Add button. Under "Websites you've added," "usps.gov" should appear.
  3. Click the Close button in the Compatibility View Settings box.
  4. The display in the browser will then correct itself. You should not need to repeat this change again.
